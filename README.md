- **Backend**: Flask
- **Frontend**: Tailwind CSS
- **PDF Processing**: PyPDF2
- **Text Summarization**: Sumy with LSA algorithm
- **File Handling**: Werkzeug

## Requirements

- Python 3.8+
- Flask
- PyPDF2
- Sumy
- NLTK
- NumPy
- SciPy
